  • Tool: One of the first examples of a RAD tool (IDE & visual window designer)

  • Language: Supported class and interface polymorphism along with interface delegation

Walled Garden Design

Pros and Cons

Category Cons Pros
Configuration
  • One more step contributing to setup fatigue
  • One-time cost that pays for itself
Type Definitions
  • Required by the compiler for external libraries
  • Incomplete or unavailable for a small number of libraries
  • Are easily installed as NPM dev dependencies
  • Are easily augmented or stubbed
  • Provide package documentation
  • Ensure correct usage of APIs
Transpilation
  • Is an extra step in the REPL cycle
  • Transforms certain code from one paradigm to another when downleveling
  • May give false sense of type safety at run-time
  • Can be mitigated by "watch mode"
  • Can generate .map files for tools
  • Offers syntax checking and error detection
  • Allows you to create a walled garden
